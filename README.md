# Market-Intelligence-and-Forecasting-Pipeline

## 📌 Project Overview
An end-to-end analytics pipeline designed to extract, process, and visualize macroeconomic indicators for strategic decision-making. This project features a central dashboard (`analystDashboard.ipynb`) that tracks market trends and generates data-driven forecasts.

## 🛠️ Tech Stack & Tools
* **Language:** Python
* **Data Integration:** REST APIs (FRED API)
* **Environment:** Jupyter Notebook

## 💻 How to Run Locally
1. Clone the repository: `git clone https://github.com/hasanbhujwala/Market-Intelligence-and-Forecasting-Pipeline.git`
2. Create a `.env` file in the root directory based on the `.env.example` file.
3. Add your personal FRED API key to the `.env` file.
4. Open and run `analystDashboard.ipynb` in your preferred Jupyter environment.
